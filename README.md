# DTPLab website

부산대학교 디지털트윈 피지컬AI 연구실 공식 홈페이지입니다. 별도 서버나 데이터베이스 없이 GitHub Pages에서 동작하는 정적 사이트로 구성했습니다.

## 배포

1. 이 폴더를 GitHub 저장소의 `main` 브랜치에 올립니다.
2. 저장소의 **Settings → Pages → Build and deployment**에서 Source를 **GitHub Actions**로 선택합니다.
3. `main` 브랜치에 변경사항을 올리면 사이트가 자동으로 배포됩니다.

## 내용 수정

- 페이지 내용: `site/index.html`
- 디자인: `site/styles.css`
- 구성원·논문 데이터와 동작: `site/app.js`
- 공유 대표 이미지: `site/og.png`
